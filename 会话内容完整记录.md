- 体态：身高 165cm，肩宽偏宽，腰腹敏感，暖调色
- 风格：通勤 Smart Casual

### 超级能力 1：穿搭决策中枢（3 场景）
1. 通勤早晨：硬约束/切换预案/置信度
2. 临时户外会议：高温/紫外线/正式场合
3. 用户抵触推荐：修正行为记录/偏好更新

### 超级能力 2：动态体态档案（3 场景）
1. 身材反馈修正：标签修正 + 档案更新
2. 视觉推断：圆肩/探颈 → 版型调整
3. 持续学习：连续拒绝 → 全局权重下降

### 超级能力 3：Wardrobe ROI（3 场景）
1. 高度相似衣物：几乎一样 → 不建议重复购买
2. 组合增量型新衣：新增搭配数 + 激活旧衣数
3. 替代与凑单：ROI 优先而非价格优先

### 极端场景测试
- 极端炎热 + 正式会议
- 用户体态快速变化
- 衣橱极简冷启动

### 验收判据
- 穿搭决策：先硬约束再美学 / 输出理由风险替代 / 场景切换 / 反馈影响
- 动态体态：单次反馈更新 / 视觉输入影响 / 连续反馈改变权重
- Wardrobe ROI：不只是相似度 / 输出组合增量 / 旧衣激活数 / 场景覆盖 / 购买建议
