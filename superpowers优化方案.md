# Superpowers 优化方案

> 基准：`superpowers评估报告_v2.md`
> 范围：冷启动、成本、端侧/云端、合规、连续反馈、核心业务逻辑、测试覆盖

---


## 优化点 1：冷启动体验过重
- 问题：新用户先录 10 分钟衣服才可用，首单转化低
- 目标：新用户 3 分钟内获得首套穿搭推荐
- 方案：
  1. 内置示例衣橱快速体验
  2. 电商订单/购买记录一键导入
  3. 社交图片批量导入 + 自动识别
- 验收：注册到首套推荐 <= 3 分钟

---


## 优化点 2：成本控制与商业化边界不清晰
- 问题：AI 成本、订阅、CPS 返点未分层，商业化风险高
- 目标：单用户月均 AI 成本 <= 0.5 元
- 方案：
  1. 基础功能免费，高级决策/社媒/日程规划订阅制
  2. 决策结果缓存 24h，命中率目标 >= 60%
  3. 模型蒸馏/量化，推理成本降 50%
  4. 夜间批量预生成次日穿搭
- 验收：订阅转化率 >= 8%

---


## 优化点 3：端侧/云端能力边界不明确
- 问题：哪些能力必须本地、哪些可云端，未明确
- 目标：无网也能查看历史衣橱和基础决策
- 方案：
  1. 端侧必跑：照片预处理、衣橱元数据缓存、离线规则决策
  2. 云端托管：向量检索、LLM 可解释生成、社媒内容池
  3. 增量同步：仅同步差异，降低流量和延迟
- 验收：离线决策成功率 >= 90%

---


## 优化点 4：合规与隐私细节不足
- 问题：社媒爬取、体态照片、记忆补全的合规边界模糊
- 目标：满足数据最小化和用户删除权
- 方案：
  1. 体态照片默认端侧处理，不上云
  2. 提供一键删除全部用户数据
  3. 社媒仅使用公开 API/授权内容
  4. 明确第三方数据使用边界
- 验收：隐私政策通过法务审核

---


## 优化点 5：连续反馈未改变全局偏好（Superpowers 验收未通过项）
- 问题：当前仅记录单次 fit_feedback，没有长期偏好权重更新
- 目标：连续 3 次拒绝修身款后，同类推荐自动降权
- 方案：
  1. 时序统计：近 7 次反馈中修身拒绝 >= 3 次 -> 权重降 20%
  2. 指数衰减平滑，避免抖动
  3. 支持用户调整学习强度（弱/中/强）
- 验收：连续拒绝后同类款排名下降 >= 2 位

---


## 优化点 6：核心业务逻辑补齐

### 模块 A：数字衣橱
- P0：真实图片上传 + 自动打标（CLIP + 分类头）
- P0：衣橱浏览筛选（品类/颜色/季节/风格）
- P1：记忆补全（购买记录/社交图片）
- P1：衣橱健康度（闲置率/组合覆盖率）
- P2：关系图谱视图

### 模块 B：穿搭决策中枢
- P0：真实衣橱候选生成（替代硬编码）
- P0：多目标优化（美学/复用/场景/偏好）
- P0：LLM 可解释 rationale
- P1：一键换一套（保留约束重生成）
- P1：穿着记录 + 决策反馈学习
- P2：穿搭日程规划

### 模块 C：买前 ROI
- P0：CLIP + Qdrant 真实相似度检索
- P0：属性级对比（颜色/品类/风格/版型）
- P0：真实组合缺口计算（图谱遍历）
- P1：旧衣激活分析（近 30 天未穿 + 可搭配）
- P1：场景覆盖提升计算
- P2：ROI 权重校准（反馈闭环 + A/B）

### 模块 D：社媒推荐
- P0：公开穿搭 API/合规内容池
- P0：同特征筛选（脸型/体型/体态/好评）
- P0：单品提取 + 购买跳转（CPS）
- P1：风格迁移验证
- P2：穿搭组合预览图

---


## 优化点 7：测试覆盖与验收标准落地
- 问题：只测 API 可用性，不测核心业务逻辑
- 目标：核心场景测试覆盖率 >= 80%
- 方案：
  1. 把 `v3_superpower_user_tests.md` 转 pytest
  2. 新增硬约束/ROI/反馈闭环单测
  3. 新增端到端集成测试
  4. 引入性能基线（100/500/1000 件衣物）
- 验收：CI 每次提交自动运行核心测试

---


## 优先级总表

| 优先级 | 优化点 | 预期价值 |
|--------|--------|----------|
| P0 | 冷启动、成本边界、端侧边界、连续反馈、核心业务逻辑、测试覆盖 | 直接提升可用性与 Superpowers 达成率 |
| P1 | 记忆补全、多目标优化、LLM、ROI 校准、社媒接入 | 提升准确性和留存 |
| P2 | 关系图谱、日程规划、风格迁移、性能优化 | 提升长期竞争力和体验上限 |

---


## 成功度量

- 冷启动到首套推荐 <= 3 分钟
- 单用户月均 AI 成本 <= 0.5 元
- 连续反馈影响推荐成功率 >= 90%
- 核心场景测试覆盖率 >= 80%
- 订阅转化率 >= 8%
